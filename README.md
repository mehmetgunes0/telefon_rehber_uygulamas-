# telefon_rehber_uygulamas-
Python programlama dili kullanılarak geliştirilmiş basit bir telefon rehberi uygulamasıdır. Kullanıcıların kişi ekleme, silme ve rehberi görüntüleme işlemlerini terminal üzerinden gerçekleştirmesini sağlar. Temel veri yapıları ve kullanıcı etkileşimini göstermek amacıyla geliştirilmiştir.
